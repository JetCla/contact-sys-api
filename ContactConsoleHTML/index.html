Contact System API version v1

The Contact API provides operations on the contact data.

Contacts

/contacts get post

get

List of Contacts which can be filtered by Last and first name. The results is paginated.

post

Add a new Contact. The Contact-info and address are not mandatory to create a contact.

get /contacts

List of Contacts which can be filtered by Last and first name. The results is paginated.

  • Request
  • Response

Query Parameters

  • offset: (integer)

    Starting page offset

    Example:

    1
  • limit: (integer - default: 100)

    Limit the number of elements on the response

    Example:

    80
  • firstName: (string)

    User first name

    Example:

    Claudio
  • lastName: (string)

    User last name

    Example:

    Desalvo
  • orderBy: (string - default: Id)

    Order by field

  • order: (one of desc, asc - default: desc)

    Order
